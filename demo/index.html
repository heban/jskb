jsKB - prosta wirtualna klawiatura w JavaScript

Info

jsKB jest wirtualną klawiaturą napisaną w języku JavaScript (bez użycia żadnego frameworka), której podłączenie sprowadza się do podania identyfikatora pola tekstowego (input oraz textarea), oraz opcjonalnego identyfikatora przycisku za pomocą którego klawiatura ma być wywoływana. Przycisk może być dowolnym elementem języka HTML. Póki co klawiatura wspiera jedynie język polski.

Przykład użycia

Pierwsze pole aktywuje klawiaturę na "focus", drugie pole wymaga kliknięcia w przycisk obok.

Umieszczanie klawiatury na własnej stronie WWW

Pliki jskb.js oraz opcjonalnie jskb.css, należy umieścić w szablonie witryny: 


<link rel="stylesheet" type="text/css" href="jskb.css" />
<script type="text/javascript" src="jskb.js"></script>

Skrypt pobiera identyfikatory pól oraz przycisków z nimi skojarzonymi, dlatego też musi się wykonać po załadowaniu strony. W sekcji <head> klawiaturę można wywołać następująco: 


<script type="text/javascript">
window.onload = function () {
	jsKB('id_pola','id_przycisku'); // lub jsKB('id_pola');
}
</script>

jsKB() jest funkcją, która przyjmuje identyfikator pola orac opcjonalnego przycisku i kojarzy z nimi klawiaturę. Identyfikatory można podać w formie łańcuchów znaków, bądź też jako konkretne elementy pobrane jakąś funkcją DOMu. Można ją zastosować do kilku pól kojarząc je z innymi przyciskami, dla przykładu: 


<script type="text/javascript">
window.onload = function () {
	jsKB('id_pola','id_przycisku');
	jsKB('id_pola2');
}
</script>

Obsługiwane przeglądarki

jsKB obsługują wszystkie wersje Firefox'a, Google Chrome, Opera (testowane na 10), Safari (testowane na 4) oraz Internet Explorer od wersji 6. Ze względu na zastosowanie w domyślnym stylu nowości z CSS3, IE6, IE7 i IE8 odstają od reszty, jednakże klawiatura na nich działa i wyświetla się poprawnie.
